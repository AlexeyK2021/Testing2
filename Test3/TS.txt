ЗАДАЧА 1.3:
Есть сервис, который хранит учетные записи и запросы. Учетная запись (УЗ) - имя, пароль. УЗ может формировать запросы сервису. Виды запросов: Get, Set. Запрос - имя аккаунта, вид запроса. Также есть УЗ админа, в которой есть возможность просмотреть статистику сайта (запросы, количество УЗ).

Постановка задачи:
Нужно реализовать сервис, который хранит массив Account[], где Account - класс для УЗ. Также УЗ должна уметь посылать запрос (достаточно просто вывести строку, что запрос дошел).
В Main дать возможность создавать УЗ и выбирать из созданных, а также для админа возможность просмотреть статистику.
Для каждого наследника Account и самого класса реализовать виртуальный метод OutputControll, который вывод строку с клавишами, которые используются для управления

Вид результатов:
Статистика: Запрос %вид запроса% от %имя аккаунта%\n
Get: возвращаю данные
Set: данные изменены

Выборы в консоли:
1. Выбор из списка созданных УЗ
2. Выбор действия после входа
